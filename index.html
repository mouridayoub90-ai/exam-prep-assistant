Activation de l'IA Dynamique (Gemini API)

Par défaut, l'application fonctionne avec notre moteur RAG & NLP local 100% hors-ligne. Pour débloquer des explications dynamiques illimitées de l'IA (en injectant les chunks de cours dans le contexte), vous pouvez renseigner votre clé API Gemini personnelle.

La clé est stockée localement dans votre navigateur (`localStorage`) et n'est jamais envoyée vers un serveur tiers.
